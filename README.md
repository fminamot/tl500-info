## OpenShiftとのネットワーク接続設定

Workstation内でターミナルを開き、以下のコマンドを実行します。

```
[student@workstation ~]$ cd /home/student/
[student@workstation ~]$ sudo ./set_tl500_network.sh --ip=10.82.1.151

```
